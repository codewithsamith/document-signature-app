# Digital Document Signature Application

This project is a full stack web application that allows users to upload PDF documents and digitally sign them.

## Features

- User Registration and Login
- Upload PDF documents
- Upload signature image
- Sign PDF documents
- Download signed documents

## Technologies Used

Frontend:
- React.js
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Libraries:
- Multer
- PDF-Lib
- JWT Authentication
